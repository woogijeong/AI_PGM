## 📝 파이썬 코드 구조화 보고서: 클래스, 모듈, 패키지

### 1. 클래스 (Class): 객체를 만들기 위한 '설계도'
**클래스**는 실생활의 물건을 만들기 위한 **설계도**나 **붕어빵 틀**에 비유할 수 있습니다. 클래스를 통해 만들어진 실제 물건을 **인스턴스(객체)**라고 부릅니다.

*   **주요 구성 요소:**
    *   **속성 (변수):** 객체가 가진 이름, 나이, 체력 등의 데이터입니다.
    *   **메서드 (함수):** 객체가 수행하는 '인사하기', '공격하기' 등의 동작입니다.
    *   **`__init__` (생성자):** 객체가 생성될 때 자동으로 호출되어 초기값을 설정해주는 특별한 함수입니다.
    *   **`self`:** 객체 자신을 가리키는 키워드로, 클래스 내부에서 자신의 속성이나 메서드에 접근할 때 사용합니다.

*   **클래스 사용 예시 (게임 캐릭터):**
    ```python
    class Hero:
        def __init__(self, name, power):
            self.name = name  # 인스턴스 변수 설정
            self.power = power

        def attack(self):
            print(f"{self.name}이(가) {self.power}의 힘으로 공격합니다!")

    # 객체 생성 및 사용
    iron_man = Hero("아이언맨", 100)
    iron_man.attack()
    ```

### 2. 모듈 (Module): 코드를 담은 '파이썬 파일'
**모듈**은 여러 개의 변수, 함수, 클래스를 모아 놓은 **하나의 `.py` 파일**입니다. 코드가 길어질 때 기능별로 나누어 저장하면 관리가 쉬워지고 다른 파일에서도 재사용할 수 있습니다.

*   **사용 방법 (`import`):**
    *   `import 모듈`: 모듈 전체를 가져옵니다.
    *   `import 모듈 as 이름`: 모듈 이름을 줄여서 사용합니다 (예: `import math as m`).
    *   `from 모듈 import 함수/클래스`: 모듈의 특정 기능만 가져와 모듈 이름 없이 바로 사용합니다.

*   **모듈 사용 예시 (수학 모듈):**
    ```python
    import math  # 파이썬 표준 라이브러리 모듈 가져오기
    print(math.pi)  # 모듈.변수 형식으로 사용
    print(math.sqrt(16))  # 16의 제곱근 출력
    ```

### 3. 패키지 (Package): 모듈을 묶은 '폴더'
**패키지**는 여러 개의 모듈을 폴더 형태로 묶어 구조화한 것입니다. 프로젝트 규모가 커져 모듈이 많아질 때 관련 있는 것끼리 분류하여 관리하기 위해 사용합니다.

*   **특징:**
    *   일반 폴더와 구분하기 위해 폴더 내부에 **`__init__.py`** 파일이 포함되어야 합니다 (파이썬 3.3 이전 버전 필수).
    *   **라이브러리(Library)**는 이러한 패키지와 모듈들을 모아둔 더 큰 집합체를 의미합니다.

*   **계층 구조 이해:**
    **라이브러리 ≥ 패키지 ≥ 모듈 ≥ 클래스/함수** 순으로 포함 관계가 형성됩니다.

### 💡 요약 및 가이드
| 구분 | 개념 | 비유 | 단위 |
| :--- | :--- | :--- | :--- |
| **클래스** | 데이터와 기능을 묶은 틀 | 설계도, 붕어빵 틀 | 코드 블록 |
| **모듈** | 클래스와 함수를 모은 파일 | 부품, 도구 상자 | `.py` 파일 |
| **패키지** | 모듈을 분류해 둔 디렉토리 | 도구함 분류 칸 | 폴더 |

**초보자를 위한 팁:**
1.  **언제 클래스를 쓰나요?** 단순히 작업을 수행하는 것이 아니라, 특정 데이터(상태)를 유지하고 관리해야 하는 객체를 만들 때 사용합니다.
2.  **모듈화의 장점:** 코드를 기능별로 독립된 파일에 저장하면 오류 수정을 최소화하고 성능을 향상시킬 수 있습니다.
3.  **표준 라이브러리 활용:** 파이썬 설치 시 기본으로 제공되는 `math`, `random` 같은 다양한 모듈과 패키지를 먼저 익히는 것이 좋습니다.
